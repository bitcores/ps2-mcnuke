# ps2-mcnuke
PS2 memory card wiping tool

This tool will erase every block on the memory card in slot 2. No questions, no verify. Will reboot to BIOS when completed or display FAIL message if failed.

Memory card can then be reformatted in memory card Browser in PS2 BIOS

May fix incorrectly flagged bad blocks and even some memory cards that don't appear in the Browser

No guarantees
